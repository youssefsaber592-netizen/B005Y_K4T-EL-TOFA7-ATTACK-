# ⚡ B005Y_K4T: EL TOFA7 ATTACK ⚡

A fast-paced 2D Arcade Platformer game built using **Godot 4**. The game challenges players to collect apples under strict time constraints, featuring dynamic game elements, screen shakes, sound design, and persistent storage.

---

## 🎮 Key Features

* **Time Attack System:** Strictly managed levels (Level 1: 30s | Level 2: 45s).
* **Save System:** High score persistence across game sessions utilizing Godot's `FileAccess`.
* **Juice & Game Feel:** Implemented dynamic screen shakes on impact/collection and animated asynchronous notification overlays.
* **Idle Player Detection:** Programmatic anti-AFK checking to prompt the player.
* **State Management:** Seamless scene reloading, progressive level tracking, and automated cleanup.

## 🛠️ Tech Stack & Concepts Applied

* **Engine:** Godot 4 (GDScript)
* **Architecture:** Node-based hierarchy, signals/events decoupling, and asynchronous flows (`await`).
* **DevOps Ready:** Clean codebase ready for automated export pipelines (CI/CD).
